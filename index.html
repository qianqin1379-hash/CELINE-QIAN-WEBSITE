Apple Style Portfolio Template

🍎 Apple-Style Portfolio Template

A modern, minimalist portfolio template built with Astro and inspired by Apple’s design aesthetic.

Live Demo: apple-style-portfolio

✨ Features

  • 🍎 Apple-style design aesthetic
  • ⚡ Built with Astro for performance
  • 🎨 TailwindCSS styling
  • 🌟 GSAP animations
  • 📱 Fully responsive
  • 🎬 Three.js 3D integration
  • ⚛️ React components support

🚀 Getting Started

Prerequisites

Install Node.js version 20+

Installation

git clone https://github.com/larry-xue/apple-style-portfolio

cd apple-style-portfolio

npm install

💻 Development

npm run dev

Then open:

http://localhost:4321

📦 Build for Production

npm run build
npm run preview

☁ Deployment

Deploy with Vercel

  1. Fork the repository
  2. Connect GitHub to Vercel
  3. Select the repository
  4. Click Deploy

Deploy with Cloudflare Pages

  1. Fork repository
  2. Login to Cloudflare
  3. Create new Pages project
  4. Build command: npm run build
  5. Output folder: dist
